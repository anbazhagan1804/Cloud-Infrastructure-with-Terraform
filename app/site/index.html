Azure + Terraform + Argo CD

Platform delivery that ships itself.

This demo application is built in Azure DevOps, pushed to Azure Container Registry, and deployed to AKS by Argo CD from the manifests in gitops/workloads/demo.

Provisioned with Terraform Networks, observability, ACR, and AKS are all created from the terraform/ folder.
Delivered with GitOps Argo CD continuously syncs this workload from Git and reconciles drift automatically.
Ready for extension Add Helm charts, ingress, cert-manager, or external secrets without changing the core flow.
